# Tripleten web_project_homeland

TripleTen Art Gallery
Descripción del proyecto

Este proyecto corresponde al Sprint 5 del programa de desarrollo web de TripleTen. Consiste en la creación de una página web adaptativa que presenta una galería de arte digital con historias y fotografías de diferentes partes del mundo. La página está diseñada para ajustarse correctamente a distintos tamaños de pantalla, garantizando una experiencia de usuario consistente en dispositivos desktop, tablet y mobile.

Tecnologías y técnicas utilizadas
HTML5
Estructuración semántica del contenido utilizando etiquetas como header, main y footer
CSS3
Estilización modular mediante la importación de múltiples archivos CSS para una mejor organización del código
Metodología BEM (Block, Element, Modifier)
Uso de clases como header**title, content**description, etc., para mantener un código escalable y mantenible
Flexbox
Implementado para la alineación y distribución de elementos en la página, especialmente en contenedores principales
Diseño adaptativo (Responsive Design)
Uso de media queries para adaptar el diseño a diferentes resoluciones (desktop, tablet y mobile)
Unidades relativas
Uso de rem y % para mejorar la escalabilidad y accesibilidad del diseño
Normalize.css
Integrado para asegurar consistencia entre navegadores
Tipografía personalizada
Uso de fuentes externas mediante @import para mantener coherencia visual
